TYPE ZOO

This project involved designing promotional posters for The Bronx Zoo. Each poster featured one animal, creatively constructed using letters that spell out the animal's name. The posters were designed in an 18" x 24" portrait format and included information about The Bronx Zoo. Real life mockups should be included.
